Determine the truth value of each of these statements if the domain of each variable consists of all real numbers.

1. $$\exist x (x^2 = 2)$$
1. $$\exist x (x^2 = -1)$$
1. $$\forall x (x^2 + 2 >= 1)$$
1. $$\forall x (x^2 \not = x)$$

Solution

1. true
1. false
1. true
1. false
