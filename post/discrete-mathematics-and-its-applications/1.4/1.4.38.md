Find a counterexample, if possible, to these universally quantified statements, where the domain for all variables consists of all real numbers.

1. $$\forall x (x^2 \ne x)$$
2. $$\forall x (x^2 \ne 2)$$
3. $$\forall x (\vert x \vert \gt 0)$$

Solution

1. x = 1
2. x = 2
3. x = 0
