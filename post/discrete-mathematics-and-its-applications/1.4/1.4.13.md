Determine the truth value of each of each of these statements if the domain consists of all integers.

1. $$\forall n (n + 1 > n)$$
1. $$\exist n (2n = 3n)$$
1. $$\exist n (n = -n)$$
1. $$\forall n (3n <= 4n)$$

Solution

1. true
1. true
1. true
1. false
    
    答案是 T 感觉有问题，负数的情况不满足
