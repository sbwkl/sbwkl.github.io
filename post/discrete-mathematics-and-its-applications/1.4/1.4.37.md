Find a counterexample, if possible, to these universally quantified statements, where the domain for all variables consists of all integers.

1. $$\forall x (x^2 \ge x)$$
2. $$\forall x (x \gt 0 \vee x \lt 0)$$
3. $$\forall x (x = 1)$$

Solution

1. Impossible
2. x = 0
3. x = 2
