Determine whether each of these conditional statements is true or false.

1. if 1 + 1 = 3, then unicorns exist.
2. if 1 + 1 = 3, then dogs can fly.
3. if 1 + 1 = 2, then dogs can fly.
4. if 2 + 2 = 4, then 1 + 2 = 3.

Solution

1. true
2. true
3. false
4. true
