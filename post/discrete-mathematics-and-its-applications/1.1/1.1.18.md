Determine whether these biconditions are true or false.

1. 2 + 2 = 4 if and only if 1 + 1 = 2.
2. 1 + 1 = 2 if and only if 2 + 3 = 4.
3. 1 + 1 = 3 if and only if monkeys can fly.
4. 0 > 1 if and only if 2 > 1.

Solution

1. true
2. false
3. true
4. false
