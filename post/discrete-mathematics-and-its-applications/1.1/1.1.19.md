Determine whether each of these conditional statements is true or false.

1. if 1 + 1 = 2, then 2 + 2 = 5.
2. if 1 + 1 = 3, then 2 + 2 = 4.
3. if 1 + 1 = 3, then 2 + 2 = 5.
4. if monkeys can fly, then 1 + 1 = 3.

Solution

1. false
2. true
3. true
4. true
