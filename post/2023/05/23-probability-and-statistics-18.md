# 《概率统计》 day 18

今天是读《概率统计》的逻辑第 18 天，学习矩，原文叫 moment。

随机变量 X 的第 k 个矩定义为 $E(X^k)$。

这个定义不管哪边都很难理解，根据知乎老哥<sup>[1]</sup>的说法，这个概念和物理学力矩有点关系。

力矩是力和力臂的乘积，杠杆两头力矩相等 ($F_1D_1 = F_2D_2$) 杠杆保持平衡。

期望类似，x 和 f(x) 的乘积，E(X) 相当于保持平衡的支点。

好像有点道理。

$E(X - \mu)^k$ 叫做第 k 个中心矩，$\mu$ 是期望。

矩为分布提供信息，比如一阶矩是期望，二阶中心矩是方差。

三阶用来计算偏度 skewness = $\frac{E[(X - \mu)]^3}{\sigma^3}$，衡量对不对称，往那边歪。

四阶用来计算峰度 kurtosis<sup>[2]</sup> = $E[(\frac{X - \mu}{\sigma})^4]$，再往后的似乎没啥资料。

Moment generating functions 简称 $m.g.f.$ 有翻译成矩母函数，也有叫动差生成函数。

根据老哥的说法<sup>[3]</sup>，矩母函数求导一次产生一个矩，像母鸡下蛋。

矩母函数定义：$\psi(t) = E(e^{tX})$，能想出这个定义的老哥也是牛逼。

对它求 n 次导就是第 n 个矩，如果矩存在的话

$E(X^n) = \psi^{(n)}(0)$

查了下资料，矩母函数的作用似乎就是求矩，理由是导数比积分容易计算。

|一些公式|前提条件|
|--|--|
|$\psi_y(t) = e^{bt}\psi_x(at)$.|$Y=aX + b$.|
|$\psi_y(t) = \prod_{i=1}^{n}\psi_i(t)$.|$Y=\sum_{i=1}^nX_i$ 且 $X_i$ 独立|
|$\psi(t)=pe^t + 1 - p$.|伯努利分布 p|
|$\psi(t)=(pe^t + 1 - p)^n$.|二项式分布 (n, p)|
