# 《洞穴奇案》读书笔记

这本书给我最大的感觉是同样的事实，可以有不一样的解读，和小时候做的阅读理解刚好相反。

这本书分前后两部分，前一部分五个观点富勒写的，后一部分九个观点萨伯写的，等待第十五个观点。

这本书虚构一个发生在纽卡斯国的案件，五名探险者困在一个危险的洞穴，最终他们通过吃掉一个同伴活了下来，然后被起诉谋杀。

案件有几个关键信息

+ 五人知道洞穴探险的危险，在协会留了洞穴的地址
+ 五人撑到二十多天时，通过无线电与救援队取得联系
+ 救援专家评估救援进度来不及，他们一定会饿死
+ 救援专家评估吃一个人可以撑到救援成功
+ 救援过程中，十名救援人员牺牲
+ 威特莫尔提议丢骰子抽签选出受害者
+ 五人反复讨论保证抽签的公平性，最终同意这种方法
+ 轮到威特莫尔，他撤回同意，并要求再等一个星期
+ 一名被告替威特莫尔投掷，威特莫尔被吃了

看着这个剧情就头皮发麻，这是希腊那帮大胡子会想的故事。

**观点 1：尊重法律条文**

法律条文明确规定「任何故意剥夺他人生命的人必须被判处死刑」因此有罪判决，但是应该获得行政赦免。

*听着有点像甩锅，有罪但是希望别人来进行行政赦免。*

**观点 2：探究立法精神**

一个人可以违反法律的表面规定而不违反法律本身。

两个理由

1. 本案不适用纽卡斯国法律，案发时「人类可能共存」的基础不存在，因此他们处于自然法状态。被告们建立新契约：通过丢骰子献祭一个人
2. 如果第一个理由不成立，本案依然可以由自我防卫杀人免责

**观点 3：法律与道德的两难**

首先觉得 2 号在扯蛋，如果这些人不受法律约束，那么他们是什么时候处于「自然状态」，什么时候又回到「文明社会状态」？

另外有先例，因饥饿盗窃被判有罪。如果饥饿不能成为盗窃食物的正当化事由，也不能成为吃人的正当化理由，不能算作正当防卫。

倾向有罪判决，但是将被处死的四人是以十个英雄为代价换来的，控告谋杀罪是一种遗憾，因此弃权。

**观点 4：维持法制传统**

不该纠结立法目的，立法可能有许多目的，纠结立法目的会掺入过多的个人因素，法官应忠实的履行自己的义务。

本案不属于自我防卫，因为自我防卫适用于当事人抵抗威胁自己生命的情形，但威特莫尔显然没有威胁被告的生命。

**观点 5：以常识来判断**

法律为人服务才有意义。报纸集团就此问题做了一个民意调查，大约九成的人认为应该宽恕被告或者给予象征性惩罚后释放。

到这里第一部分就结束了，两赞成两反对一弃权。

**观点 6：撇开己见**

对立法机关而言，法律和道德不可分割。对司法机关而言，法律和道德相互独立。

公众仅仅考虑道德而要求宽恕被告，法律无关同情，守护法律是法官的职责，依照法律被告有罪。

紧急避难抗辩不成立，理由有六

1. 滥用紧急避难将破坏法治
2. 饥饿不能构成紧急避难（冉阿让案）
3. 减轻饥饿并非只有杀人一种选择，比如等第一个人饿死后再吃尸体，或者吃「小零食」，或者再等几天
4. 制造危害者不能受惠于紧急避难，他们明确知道山洞探险有危险，他们是自愿将自己暴露在风险中，不能以自愿面对的危险为由杀掉别人
5. 被告应对危机不足，没有带够足够的粮食
6. 选择被害人有欠公平，因为被害人撤回了同意

**观点 7：判案的酌情权**

紧急避难抗辩的内在法律原则是，由于紧急避难而实施犯罪的人没有犯罪意图，所以不应该受到惩罚。

他们有意的选择了掷骰子的方法，也有意的杀害被选出来的人，但他们并非有意的选择了威特摩尔。

紧急避难抗辩成立，理由也有六

1. 本案紧急避难的确信合理，他们被困二十天之久，营救至少还需要十天，没有食物活下来只有「极小的可能性」，吃掉一人则至少可以活十天，征求杀掉一人的法律和道德意见时，没有得到回应
2. 本案中饥饿可作为一种紧急避难，冉阿让案被告有其他选择
3. 当时情形下杀人是生存的唯一选择，他们至少要生存十天之久，吃「小零食」基本不可能活下来，而且更要忍受折磨
4. 身处危境不是被困者过错，他们知道这是危险的活动，带了无线电，给协会留信息。这都不行，那屋子起火打破玻璃逃生，能不能算紧急抗辩。每个人都知道建筑物有起火风险，还住在里面
5. 他们带了至少二十天的食物，食物匮乏并非疏忽大意造成
6. 受害人的同意无关紧要，丢骰子被论证是公平的

和 6 号针锋相对

惩罚被告有悖法律的目的，惩罚他不会阻止将来也不幸陷入相同困境的人不去做相同的事。

**观点 8：一命换多命**

我们珍视生命，而且总倾向于更多的人而不是更少的人在悲剧性事故中存活下来。

法律允许预防性杀人，自我防卫杀人不是谋杀，有正当性。

一命换多命是一项划算的「交易」，倾向于更多的人在悲剧性事故中活下来。

选择杀人好过等待自然死亡，等待第一个人自然死亡，这是不公平的，这种方式等于把目标锁定在最虚弱、多病、受伤最重的人身上。

平等地承担死亡是公平的，公平的抽签使得牺牲者的同意是不必要的。如果威特莫尔撤回同意退出抽签，那么剩下的人被选中的概率从 1/6 增加到 1/5 进而又有人退出，直到抽签计划被废弃。

这是 FZ 的切嗣爸爸吧，一炮把自己师傅轰下来，打的冬木市遍地废墟，最后只救了土狼。

**观点 9：动机与选择**

人们可以有意识的做某种行为，同时又违背自己的意愿。被告杀人是唯一的求生选择，被告有意地杀了人，但是那并不意味着被告的动机是杀人。

没有目的的惩罚毫无意义。面对同样的困境，探险者面临着死亡的威胁：要么饿死，要么被处死，他们还是会选择杀死一人，然后碰运气辩解。


**观点 10：生命的绝对价值**

在法律看来，每一个生命都是极其崇高和无限珍贵的。这让每个生命具有平等的价值，没有哪一个生命能够优先于其他生命。

任何牺牲都必须是自愿的，否则就是侵犯了法律所确认的生命平等和神圣尊严。

忍受不正义好过实施不正义。被告是被诱惑违反法律的，他们太脆弱以至于无法抵抗这种诱惑。他们发现选择不服从法律比选择服从法律更有利。

受害人的同意至关重要，否则一个患有肾脏疾病的将死之人为了活命，就可以通过抽签的方式决定取走一个健康人的肾脏移植，而不需要得到他的同意。

杀人永远不是「划算」的交易。无法衡量，按人头，按贡献还是啥的。

承认生命神圣原则，有罪但免于死刑。

**观点 11：契约和认可**

我们遵守法律建立在我们遵守它的承诺上面，尽管这种承诺可能是默示的。

1. 被害人撤回同意的行为不容忽视
2. 被告须为自己的行为负责，他们为了娱乐自愿走进一个布满危险的自然环境中
3. 被害人有自我防卫的权利。如果探险者们处于紧急避难杀了威特莫尔，那威特莫尔保护自己就是错的，但显然不是这样

无罪判决可能会导致悲剧重复发生，强烈的求生欲会克服杀人的不情愿。

**观点 12：设身处地**

假如法官发现自己在惩罚一个不比自己坏的人，它应该辞职。假如我在山洞里，我会不会杀人，会。

本次判案有必要抛弃客观性的面具，依靠无任何矫饰的自我意见来断案。

**观点 13： 判决的道德启示**

如果刑法的首要社会功能就是保护公民们免受犯罪所带来的特定种类的伤害，那对心理性免责事由的继续承认会加剧问题，而不会有助于问题之解决。，但是它们都是无限集合。

严格惩罚犯罪是预防犯罪最有效的手段，惩罚犯罪是对理性犯罪的威慑，废除免责事由有助于减少犯罪。

**观点 14：利益冲突？**

案件疑难意味着法律帮不上忙，欠缺法律规定意味着自由裁量权无可避免，自由裁量权意味着超乎法律之外的道德标准必须纳入案件解决的过程中。

法律允许负责地适用自由裁量。

最后法官以无线电电池利益相关回避本案。

*我感觉是随便找了个理由。*
