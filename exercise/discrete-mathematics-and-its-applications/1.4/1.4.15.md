Determine the truth value of each of these statements if the domain for all variables consists of all integers.

1. $$\forall n (n^2 >= 0)$$
1. $$\exist n (n^2 = 2)$$
1. $$\forall n (n^2 >= n)$$
1. $$\exist n (n^2 < 0)$$

Solution

1. true
1. false
1. true
1. false
