Determine the truth value of each of these statements if the domain consists of all real numbers.

1. $$\exist x (x^3 = -1)$$
1. $$\exist x (x^4 < x^2)$$
1. $$\forall x ((-x)^2 = x^2)$$
1. $$\forall x (2x > x)$$

Solution

1. true
1. true
1. true
1. false
